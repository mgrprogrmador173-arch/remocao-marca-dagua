# Removedor de Marca d'Água de Vídeo (Simulação)

Este é um projeto simples de um aplicativo web que simula a remoção de marcas d'água de vídeos. Ele permite que você faça upload de um vídeo e visualize uma simulação do processo de remoção.

## Como usar

1. Faça o upload de um vídeo clicando na área designada.
2. Clique no botão "Simular Remoção de Marca d'Água" para ver a simulação.

## Como implantar no GitHub Pages

1. Crie um novo repositório no GitHub.
2. Clone o repositório localmente.
3. Copie os arquivos deste projeto para o repositório clonado.
4. Faça commit e push para o GitHub.
5. Vá em "Settings" > "Pages" no seu repositório e selecione a branch `main` como fonte.
6. Seu aplicativo estará disponível em `https://<seu-usuario>.github.io/<nome-do-repositorio>/`.

## Observações

- Este aplicativo é apenas uma simulação e não remove marcas d'água de fato.
- Para remoção real, utilize ferramentas especializadas como FFmpeg, Adobe After Effects, ou serviços online.
- Sempre respeite os direitos autorais ao manipular vídeos.
